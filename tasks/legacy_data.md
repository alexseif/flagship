# Legacy Data Extraction

This document contains the exact names of menus, sliders, and their placements extracted from the original `db207080_eka` database. The builder should use these names when creating placeholder blocks and menus.

## Navigation Menus (by Language)

*   **Footer:** `Footer Greek Menu` (ID: 21)
*   **Sub-navigation (In-page):**

## Slider Placements

When adding placeholder Native Gutenberg Carousel/Gallery blocks, use these references for the user to import later.

### Global / Structural
*   **Homepage (`front-page.html`):** Uses LayerSlider ID 6 ("Homepage slider").
*   **News Archive / Page:** Uses LayerSlider ID 6 ("Homepage slider").

### Specific Inner Pages
The following pages require a native slider placeholder block. The exact pages span across Greek, English, and Arabic translations.

1.  **Staff / Στελέχωση / العاملين:** Requires `staff` slider.
2.  **Community Lounge / Κοινοτικό Εντευκτήριο / استراحة الجمعية:** Requires `GREEK-CLUB` slider.
3.  **Cemeteries / Κοιμητήρια / المقابر:** Requires `Κοιμητήρια` slider.
4.  **Cemeteries Conservation / Συντήρηση Κοιμητηρίων / المقابر:** Requires `cemeteries-maintenance` slider.
5.  **Music Museum / Μουσείο Μουσικής / متحف موسيقي:** Requires `music-museum` slider.
6.  **Science Museum / Μουσείο και Εργαστήριο Θετικών Επιστημών / متحف ومختبر:** Requires `science-museum` slider.
